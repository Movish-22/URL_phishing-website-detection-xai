# Phishing Website Detection using Machine Learning

Student: Monika Thapa  
University: Auckland University of Technology

## Project Overview

This project implements machine learning models to detect phishing websites using the UCI Phishing Websites Dataset (2014).

Models implemented:
- Logistic Regression
- Decision Tree
- Random Forest

## Explainable AI

To improve model transparency, two explainability techniques are used:

- SHAP (SHapley Additive exPlanations)
- LIME (Local Interpretable Model-agnostic Explanations)

These methods help explain why the model predicts a website as phishing or legitimate.

## Dataset

UCI Phishing Websites Dataset (2014)  
Total samples: 11,055  
Features: 30
